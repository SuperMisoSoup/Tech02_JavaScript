じゃんけんクエスト

## ②課題内容（どんな作品か）

- じゃんけんでバトルしながら冒険を進めるゲーム
- バトルでは各手に設定された攻撃力で相手のHPにダメージを与える
- スタート→PC→ボーナス→ボス　の4ステージ

## ⑤工夫した点・こだわった点

- ただのじゃんけんだけでなくゲーム性を持たせた
- 前のステージの自分のHPを維持するようにした
- ステージが終わった後は処理を中止させるようにした
- ステージが終わった後に結果によって遷移先が異なるボタンを表示させるようにした
